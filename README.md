# NoNPC

App web de misiones cortas para salir del piloto automático y dejar de ser un NPC. Recibís desafíos aleatorios (desde simples hasta incómodos), los completás en la vida real y ganás XP para subir de nivel.

## 🧠 Concepto
NoNPC no es productividad tradicional. Es incomodarte un poco para generar experiencias reales y romper la rutina.

## ⚙️ Funcionalidades
- Misiones aleatorias
- Sistema de XP
- Niveles
- Historial de misiones
- Dificultades (Fácil, Media, Hardcore)

## 🕹️ Uso
1. Abrís la app  
2. Aceptás una misión  
3. La completás en la vida real  
4. Ganás XP  

## 🧩 Tecnologías
- HTML
- CSS
- JavaScript (vanilla)

## 📦 Estado
MVP funcional en desarrollo

## 💡 Futuro
- Login de usuario
- Misiones diarias
- Compartir misiones
- Ranking global
